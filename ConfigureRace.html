Configure Race

Configure the event in the "Config" Tab

    1. Open xcscoreboard in standalone mode (either as a new race file or an existing race).

    2. Input Race Information (Name/Location/Date).

    3. If using standard team size and tiebreaker rules, leave "Team Size Properties" at the defaults.

    4. Optionally “Activate” an online Scoreboard (note this requires a purchase and can only be done within two weeks of the event). Activation can be done any time within two weeks of race date. It can even be performed on the morning of race day (since it is automatic and immediate).
