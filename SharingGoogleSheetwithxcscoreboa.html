Sharing Google Sheet with xcscoreboard

Google Drive Walkthrough -- Sharing a Spreadsheet with xcscoreboard


Note that xcscoreboard can import/export to/from existing google sheets (it doesn't have the ability to create new sheets). Therefore you must grant the xcscoreboard service account email, access to each desired spreadsheet (or to the owning folder).


Here are the steps:


  1. Open xcscoreboard in standalone mode and with a good, solid, internet connection.

  2. If not done already, set the location of the Google Key File – Select Credentials -> Set Location of Google Key File and drill into the folder containing the JSON file you downloaded when you created the Google Service Account.

  3. Select Credentials -> Show Google Key File Service Account Email Address. Press the OK button to copy the service account email address to the clipboard.

  4. Open Google Drive and select the spreadsheet or folder that you wish to share with xcscoreboard.

  5. Click the share button on the toolbar.


  1. In the entry box labeled "People", paste in the service account email address and verify "Can Edit" is selected in the associated drop down.

  2. Now xcscoreboard can import or export to the google spreadsheet on which you just configured sharing.
