Integrating xcscoreboard with Google Sheets

Google Sheets Integration


Before you can import or export to Google Spreadheets in your Google Drive, you need to first create a service account in the Google Developers Console and download a JSON file.


After downloading the JSON file, click the "Credentials" menu option and pick "Set Location of Google Key File" to select the JSON file that was downloaded. This step only needs to occur once for any and all race files.
