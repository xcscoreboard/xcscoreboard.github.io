Racers Page

The Racers Page is where a team’s registered racers are entered and each assigned a Group and Bib#. Quick entry has been configured to speed manual entry. When there is an empty row below the row with the active entry, when the <enter> key is pressed, the active row’s Group, Team Name, and the next consecutive Bib# is copied to the next row. Additionally, the next row’s Racer Name entry is activated for input so all you need to enter is the racer’s name … if the Group or Team Name is different from the above, simply tab to that cell and make the change.


    • The Racers sheet can also be populated by importing directly from a CSV file, Excel file, or online Google Spreadsheet with the same field order as the Racers Sheet, i.e., Racer Name, Group, Team Name, Bib #. Simply use the Import menu to select the desired import method and select the desired file and worksheet.

    • Note that imported data begins in row 1 and overwrites subsequent row data up to the number of rows that exist in the source file.

    • The Racers sheet can be sorted by any of the four fields by clicking the desired column header field name.

    • After all Racers have been entered, click the Team Sheets link to open the list in a web browser. This list is formatted with page breaks between each team, so that when printed, each team will be printed in its own set of pages. Print and add these pages to each team’s race packet so they will know their bib assignments.
