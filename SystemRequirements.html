System Requirements

Welcome ››

Cross Country Scoreboard's recommended system configuration includes:

    • Windows XP, Windows Vista, Windows 7, Windows 8 or Windows 10
    • 2 GB of RAM
    • 80MB of free disk space
    • 1024x768 screen resolution, but higher is better
    • Additionally it is recommended that during the race, to have two computers, both connected to the same WIFI hotspot (even when using a timing system)
    • To enable online scoring, the timing computer must have internet access at the race site
    • If importing/exporting to Excel, then MS Excel needs to be installed
    • If importing/exporting to Google Spreadsheets, then a Google OAuth2 service account token in the form of a JSON file needs to be downloaded. For more information see Integrating with Google Sheets
