Register Runners

Enter/Register runners in the "Racers" Tab

    1. Assign unique bib#'s to each racer.
      Note that xcscoreboard supports multiple groups racing at a single event, e.g., your group names night be Varsity Girls, Varsity Boys, JV Girls, JV Boys, MS Boys, MS Girls.

    2. Make sure group name spelling and case is consistent across all racers – xcscoreboard is case sensitive, i.e., JV Boys and JV boys are recognized as two different groups.

    3. As well as manual entry, Cross Country Scoreboard also supports importing racer registrations direct from Excel, CSV file, or Google Sheets. Personally, I recommend a strategy of having participating team coaches entering racers in a shared Google sheet for import. For more information on this process see Strategy for Race Registration.

    4. If using a timing system (currently Ipico is the only supported timing system), then enable chip timing (menu -> Timing ->  Enable Chip Timing) and populate the Chip/Bib Mapping table. This can be done manually or by importing the mapping data from a CSV mapfile, Excel, or Google Sheets
